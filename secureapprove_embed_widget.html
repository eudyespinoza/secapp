SecureApprove Embedded Biometrics

Widget HTML para embeber en sitios de clientes. Ejecuta login biometrico, crea solicitud de aprobacion y confirma/rechaza con auditoria WebAuthn.

1) Configuracion API

Nota: el token debe pertenecer a un usuario autenticado con permisos de administracion del tenant.

2) Login Biometrico (usuario actor)

Este paso consume /{lang}/auth/webauthn/login/options y /{lang}/auth/webauthn/login/verify.

3) Solicitud de Aprobacion + Confirmacion

Se envia contexto completo de tenant + actor + usuario sujeto en el campo context.
